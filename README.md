# ProjetoGameIF

Este é um jogo desenvolvido com o framework Phaser, como parte da matéria de Tópicos Avançados 1 do curso de Análise e Desenvolvimento de Sistemas (ADS) do IFSP-CJO.

Seu nome ainda não está definido, mas ele é inspirado no jogo de fliperama *Crazy Climber*, lançado em 1980. O jogador controla um monge budista conhecido como Mestre Zen, que busca subir uma montanha para organizar um retiro espiritual em seu topo enquanto várias ferramentas tecnológicas (como drones e robôs) tentam derrubá-lo.
